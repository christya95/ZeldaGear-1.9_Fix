# 🧝 ZeldaGear 1.9 (Fixed some bugs)

Drop-in Zelda Gear mod packaging for a Hytale server root. 🎒✨

## 📦 What’s included

- `mods/ZeldaGear-1.9.jar` (the mod)
- `mods/ZeldaGear-1.9/` (extracted override folder for tweaks/fixes)

## 🛠️ Install (server)

1. 🛑 Stop the server
2. 📁 Copy `mods/ZeldaGear-1.9.jar` into your server `mods/` folder
3. 🚀 Start the server (restart recommended after any `mods/` changes)
